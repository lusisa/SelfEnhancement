# Phase 1: 数据结构与算法打卡指南

作为 AI 工程师，虽然不像纯算法工程师那样天天手撕红黑树，但扎实的逻辑思维是必不可少的。

## 核心数据结构（必懂）
*   **数组 (Array)**：内存连续，随机访问快，插入删除慢。
*   **链表 (Linked List)**：内存分散，插入删除快，随机访问慢。
*   **栈与队列 (Stack & Queue)**：LIFO 与 FIFO。理解广度优先搜索 (BFS) 与深度优先搜索 (DFS) 的基础。
*   **哈希表 (Hash Table / Dictionary)**：O(1) 的查询时间，写代码最常用的结构。
*   **树 (Tree)**：特别是二叉树、二叉搜索树。

## 核心算法思想
*   **双指针 (Two Pointers)**
*   **滑动窗口 (Sliding Window)**
*   **二分查找 (Binary Search)**
*   **深度优先/广度优先搜索 (DFS/BFS)**

## LeetCode 刷题计划 (每天 1-2 题)
不要追求难题，把“简单”和“中等”题做熟练。
建议按照分类来刷，例如经典的 **"LeetCode 101"** 或 **"代码随想录"**。

**本阶段目标**：
独立完成 30-50 道 LeetCode 简单/中等题，能够清晰地说出解题思路和时间/空间复杂度。
